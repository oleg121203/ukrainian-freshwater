# AquaFarm Freshwater Prawn Website

Створити сучасний веб-сайт для фермерського господарства, що спеціалізується на вирощуванні річкових раків та креветок Macrobrachium rosenbergii, з інноваційним 3D-інтерфейсом та повноцінним функціоналом інтернет-магазину.

**Experience Qualities**:
1. **Преміальність** - відчуття якості та ексклюзивності продукції через елегантний дизайн
2. **Інтерактивність** - захоплюючий досвід через 3D-візуалізацію та анімації
3. **Довіра** - надійність та прозорість через відгуки, історії та детальну інформацію

**Complexity Level**: Complex Application (advanced functionality, accounts)
Потребує складної архітектури для підтримки багатомовності, 3D-візуалізації, системи лояльності, інтернет-магазину та користувацьких акаунтів.

## Essential Features

**3D Prawn Visualization & Navigation**
- Functionality: Інтерактивна 3D-модель малайзійської креветки з можливістю обертання
- Purpose: Створення унікального першого враження та демонстрації продукції
- Trigger: Завантаження головної сторінки
- Progression: Завантаження моделі → можливість обертання → плавна анімація виходу меню → навігація
- Success criteria: Модель завантажується за 3 секунди, плавне обертання, меню з'являється природно

**Bilingual Support (UA/EN)**
- Functionality: Перемикання між українською та англійською мовами з пріоритетом української
- Purpose: Обслуговування місцевих та міжнародних клієнтів
- Trigger: Кнопка перемикання мови в хедері
- Progression: Клік → зміна всього контенту → збереження вибору → оновлення URL
- Success criteria: Миттєве перемикання, збереження вибору між сесіями

**E-commerce Store**
- Functionality: Каталог продукції, кошик, оформлення замовлень, система платежів
- Purpose: Продаж свіжих креветок та супутніх товарів
- Trigger: Натискання "Магазин" в меню
- Progression: Перегляд каталогу → додавання в кошик → оформлення → оплата → підтвердження
- Success criteria: Інтуїтивний процес покупки, збереження кошика, розрахунок доставки

**Photo Gallery**
- Functionality: Красива галерея з фото ферми, процесу вирощування та продукції
- Purpose: Візуальна демонстрація якості та процесу виробництва
- Trigger: Натискання розділу "Галерея"
- Progression: Відкриття галереї → перегляд мініатюр → збільшення фото → слайдшоу
- Success criteria: Швидке завантаження, плавні переходи, адаптивний дизайн

**Customer Loyalty System**
- Functionality: Накопичення балів, знижки, персональні пропозиції
- Purpose: Утримання клієнтів та стимулювання повторних покупок
- Trigger: Реєстрація/авторизація користувача
- Progression: Реєстрація → накопичення балів → отримання знижок → персональні пропозиції
- Success criteria: Прозора система нарахування, зручний особистий кабінет

## Edge Case Handling
- **Повільне інтернет-з'єднання**: Прогресивне завантаження, fallback для 3D-моделі
- **Старі браузери**: Альтернативний інтерфейс без 3D, graceful degradation
- **Мобільні пристрої**: Спрощена 3D-взаємодія, touch-friendly інтерфейс
- **Помилки платежів**: Чітке повідомлення, можливість повторної спроби
- **Відсутність товару**: Система сповіщень про надходження

## Design Direction
Дизайн має викликати відчуття свіжості, природності та преміальності - поєднання сучасних технологій з природними матеріалами, використовуючи багатий інтерфейс з деталізованими елементами для демонстрації якості продукції.

## Color Selection
Analogous (сусідні кольори на колірному колі)
Використання відтінків блакитного та зеленого для асоціації з водним середовищем та свіжістю, з акцентами теплого помаранчевого для контрасту.

- **Primary Color**: Глибокий морський блакитний `oklch(0.5 0.15 240)` - надійність та професіоналізм
- **Secondary Colors**: М'який аквамарин `oklch(0.7 0.1 200)` - свіжість води, природний зелений `oklch(0.6 0.12 150)` - екологічність
- **Accent Color**: Теплий помаранчевий `oklch(0.65 0.18 50)` - привертання уваги до CTA та важливих елементів
- **Foreground/Background Pairings**: 
  - Background (White `oklch(0.98 0 0)`): Dark text `oklch(0.15 0 0)` - Ratio 15.8:1 ✓
  - Primary (Deep Blue `oklch(0.5 0.15 240)`): White text `oklch(0.98 0 0)` - Ratio 7.2:1 ✓
  - Accent (Warm Orange `oklch(0.65 0.18 50)`): White text `oklch(0.98 0 0)` - Ratio 4.9:1 ✓

## Font Selection
Типографіка має поєднувати сучасну читабельність з природною теплотою - використання Inter для чистого, професійного вигляду з додаванням Merriweather для заголовків, що створюють відчуття ремісничої якості.

- **Typographic Hierarchy**: 
  - H1 (Hero Title): Merriweather Bold/48px/tight letter spacing
  - H2 (Section Headers): Merriweather Bold/32px/normal spacing  
  - H3 (Subsections): Inter SemiBold/24px/normal spacing
  - Body Text: Inter Regular/16px/1.6 line height
  - Navigation: Inter Medium/14px/wide letter spacing

## Animations
Анімації мають імітувати плавні рухи у воді та природні переходи, створюючи відчуття органічності та життя, з особливим акцентом на 3D-візуалізації як центральному елементі досвіду.

- **Purposeful Meaning**: Рухи нагадують течію води, плавність креветок, органічні переходи
- **Hierarchy of Movement**: 3D-модель як фокусна точка, меню з'являється як бульбашки, карточки товарів плавно піднімаються

## Component Selection
- **Components**: Card для товарів та контенту, Dialog для деталей товару, Carousel для галереї, Form для замовлень, Tabs для категорій, Button з варіантами для різних дій, Badge для статусів та знижок
- **Customizations**: 3D Canvas компонент (Three.js), Language Switcher, Interactive Map, Rating System, Loyalty Points Display
- **States**: Hover ефекти з легким підйомом, loading стани з wave анімацією, success стани з checkmark анімацією
- **Icon Selection**: Phosphor icons з nature та commerce наборів - Fish, ShoppingCart, Star, MapPin, Heart
- **Spacing**: Консистентні відступи 4/8/16/24/32px з Tailwind spacing scale
- **Mobile**: Mobile-first підхід з колапсуючою навігацією, стекованими карточками, спрощеною 3D-взаємодією через touch жести
